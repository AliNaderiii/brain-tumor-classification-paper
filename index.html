Convolutional Neural Network and Channel Attention Mechanism for Multiclass Brain Tumor Classification

Authors: Ali Naderi, Akbar Asgharzadeh-Bonab, Farid Ahmadi, Hashem Kalbkhani

Published in: Complexity (Wiley), June 2025

DOI: 10.1155/cplx/1644859

Abstract

This paper presents a deep learning framework using a convolutional neural network (CNN) integrated with channel attention mechanisms for multiclass classification of brain tumors in MRI images. The model achieves high accuracy and robust performance, showing promising results for early tumor detection and medical diagnosis support.

Keywords

  • Brain Tumor Classification
  • Convolutional Neural Network
  • Channel Attention
  • Deep Learning
  • Medical Image Analysis
